Inkscape Grid Maker

Inkscape extension to generate grids: guides defining columns and gutters.

Inkscape grid maker

This Inkscape extension will generate a grid (guides) with equal width columns and gutters, or rows and gutters.

In graphic design, a grid is a structure (usually two-dimensional) made up of a series of intersecting straight (vertical, horizontal, and angular) or curved guide lines used to structure content. The grid serves as an armature on which a designer can organize graphic elements (images, glyphs, paragraphs) in a rational, easy to absorb manner. - Wikipedia

This extension was conceived as a tool for making grids when designing websites in Inkscape. The guides are therefore positioned using (rounded) pixels. Not tested with other units.

It is an alternative to positioning paths and converting them to guides, or to using the Grids in the Document Properties.

Features

  • Define number of columns, column width and gutter width to generate a grid;
  • Gutter width can be set to zero;
  • Choose to align the grid in relation to the page: left aligned, centered or right aligned;
  • On top of this alignment, you can set a left offset (for columns), or top offset (for rows) - this offset distance can be negative;
  • Same with rows - horizontal guides with gutters and offset;
  • Option to delete all existing guides before generating the new guides;
  • Live preview to test different widths

How to install the extension

  1. Download the zip archive to your computer;
  2. Unzip (extract) the archive on your computer;

  3. Open it. In the folder grid-maker-extension, you wil find two files: grid_maker.inx and grid_maker.py. Copy these two files (and only these two files) into your Inkscape extensions folder:

  • on Windows: "C:\Program Files\Inkscape\share\extensions"
  • on Linux: " ~/.config/inkscape/extensions" (where "~" is /home/yourusername/)
  • on OS X: "/Applications/Inkscape.app/Contents/Resources/extensions"

Restart or open Inkscape.

Usage

You will find the extension under menu Extensions > Render > Guides grid maker (not to be confused with Guides creator, or Grid).

Change the settings, click Apply. Close. Done.

Or change the settings, select Live Preview, adjust settings, click Apply. Close. Done.

That's it.

Tips

  • You can generate several grids, clicking Apply for every grid - just be sure to uncheck "Delete existing guides".
  • Need guides in the middle of your gutters? After generating the columns, generate a new grid with gutters set to zero, column width set to [original column width + gutter width].
  • You can generate several grids side by side, using the large offset. Play with it.
  • No need for outer gutters/guides? Just delete them manually.
  • Need a baseline grid in addition to the generated columns ? For that it might make more sense to make a new Grid under File > Document Properties > Grids instead of this extension - setting Spacing X to 2000, Spacing Y to 14 for example. You can then toggle the guides and the baseline grid independently with shortcuts.

Other grid related tools for web design

Online grid generators and previewers that might be useful in combination with the Grid Maker extension (I sometimes use some of them to get the right widths before using my extension):

  • grid calculator
  • variable grid system - this tool can actually generate css classes, just input the same numbers you used for designing your grid (fluid and fixed css grid)
  • grid calculator and generator - set a fixed total width, give a range of column/gutter width, get all possible combinations.

The Grid Maker extension generates guides, helpful for example in designing websites. However, it does not create the css - you will have to do that yourself. Some ready made tools and frameworks that can help with grid based websites (in addition to learning css!): variable grid system, bootstrap, foundation, susy , and many others... or build your own...
